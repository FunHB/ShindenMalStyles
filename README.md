# ShindenMalStyles

css Styles to Shinen anime list to make your list look like that from MyAnimeList

To use these styles download file main.css from this repository and upload to your cloud storage like Google Drive. Next paste raw link to this file to your Shinden anime list settings under "Own CSS" option to the text field.
